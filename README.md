As this app is not hosted, below are the instructions to run it locally:
1. Clone repository
2. CD to folder on your local machine
3. CD to client/rock_the_vote
4. RUN 'npm i' --To install required dependencies
5. RUN 'npm start'
6. Open a second terminal
7. CD to server
8. RUN 'npm i' --To install required depencies
9. RUN 'nodemon server.js'
10. Login and test drive the app.

This app was created using: REACT.JS, NODE.JS, EXPRESS.JS WITH MONGOOSE, and MONGODB
